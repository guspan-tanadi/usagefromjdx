## Global Flags
<!-- [USAGE] global_flags -->
- `-C --cd <DIR>`: Change directory before running command
- `-q --quiet`: Suppress non-error messages
- `-v --verbose...`: Show extra output (use -vv for even more)
- `-y --yes`: Answer yes to all confirmation prompts
